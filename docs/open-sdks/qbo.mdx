## Integrating with Qbo OpenSDK 

Welcome to the Qbo OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Qbo OpenSDK allows you to easily integrate with the Qbo API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt.

## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {qboSdkDef} from '@opensdks/sdk-qbo'

    const qbo = initSDK(qboSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['QBO_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    
    const res = await qbo.GET('/account/{id}')
```
