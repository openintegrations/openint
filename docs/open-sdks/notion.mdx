## Integrating with Notion OpenSDK 

Welcome to the Notion OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Notion OpenSDK allows you to easily integrate with the Notion API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt. 

### Available Methods:

| Method | Path | Description |
|--------|------|-------------|
| **GET** | /v1/users/{user_id} | Retrieve a user
| **GET** | /v1/users | List all users
| **POST** | /v1/databases/{database_id}/query | Query a database
| **POST** | /v1/search | Search by title
| **GET** | /v1/blocks/{block_id}/children | Retrieve block children
| **PATCH** | /v1/blocks/{block_id}/children | Append block children
| **GET** | /v1/blocks/{block_id} | Retrieve a block
| **PATCH** | /v1/blocks/{block_id} | Update a block
| **DELETE** | /v1/blocks/{block_id} | Delete a block
| **GET** | /v1/pages/{page_id} | Retrieve a page
| **PATCH** | /v1/pages/{page_id} | Update page properties
| **POST** | /v1/pages | Create a page
| **POST** | /v1/databases | Create a database
| **PATCH** | /v1/databases/{database_id} | Update a database
| **GET** | /v1/databases/{database_id} | Retrieve a database
| **GET** | /v1/pages/{page_id}/properties/{property_id} | Retrieve a page property item
| **GET** | /v1/users/me | Retrieve your token's bot user
| **GET** | /v1/comments | Retrieve comments
| **POST** | /v1/comments | Create comment
| **POST** | /v1/oauth/token | Create a token


## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {notionSdkDef} from '@opensdks/sdk-notion'

    const notion = initSDK(notionSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['NOTION_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    // Retrieve a user
    const res = await notion.GET('/v1/users/{user_id}')
```
