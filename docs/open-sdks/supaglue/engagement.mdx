## Integrating with Supaglue OpenSDK 

Welcome to the Supaglue OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Supaglue OpenSDK allows you to easily integrate with the Supaglue API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt.

|--------|------|-------------|

| **GET** | /accounts | List accounts
| **PARAMETERS** | /accounts | 
| **POST** | /accounts/_search | Search accounts
| **PARAMETERS** | /accounts/_search | 
| **GET** | /accounts/{account_id} | Get account
| **PATCH** | /accounts/{account_id} | Update account
| **PARAMETERS** | /accounts/{account_id} | 
| **POST** | /accounts/_upsert | Upsert account
| **PARAMETERS** | /accounts/_upsert |

## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {supaglueSdkDef} from '@opensdks/sdk-supaglue'

    const supaglue = initSDK(supaglueSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['SUPAGLUE_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    // Create account
    const res = await supaglue.POST('/accounts')
```
