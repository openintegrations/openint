## Integrating with Lever OpenSDK 

Welcome to the Lever OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Lever OpenSDK allows you to easily integrate with the Lever API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt. 

### Available Methods:

| Method | Path | Description |
|--------|------|-------------|
| **GET** | /postings/{id} | 
| **GET** | /postings | 
| **GET** | /opportunities/{id} | 
| **GET** | /opportunities/{id}/offers | 
| **GET** | /opportunities | 
| **GET** | /contacts/{id} |

| **GET** | /stages | 
| **GET** | /archive_reasons | 


## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {leverSdkDef} from '@opensdks/sdk-lever'

    const lever = initSDK(leverSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['LEVER_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    
    const res = await lever.GET('/postings/{id}')
```
