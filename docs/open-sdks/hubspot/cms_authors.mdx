## Integrating with Hubspot OpenSDK 

Welcome to the Hubspot OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Hubspot OpenSDK allows you to easily integrate with the Hubspot API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt. 

### Available Methods:

| Method | Path | Description |
|--------|------|-------------|
| **GET** | /cms/v3/blogs/authors/{objectId} | Retrieve a Blog Author
| **DELETE** | /cms/v3/blogs/authors/{objectId} | Delete a Blog Author
| **PATCH** | /cms/v3/blogs/authors/{objectId} | Update a Blog Author
| **POST** | /cms/v3/blogs/authors/batch/update | Update a batch of Blog Authors
| **POST** | /cms/v3/blogs/authors/multi-language/detach-from-lang-group | Detach a Blog Author from a multi-language group
| **PUT** | /cms/v3/blogs/authors/multi-language/set-new-lang-primary | Set a new primary language
| **POST** | /cms/v3/blogs/authors/batch/archive | Delete a batch of Blog Authors
| **POST** | /cms/v3/blogs/authors/batch/create | Create a batch of Blog Authors
| **POST** | /cms/v3/blogs/authors/batch/read | Retrieve a batch of Blog Authors
| **GET** | /cms/v3/blogs/authors | Get all Blog Authors
| **POST** | /cms/v3/blogs/authors | Create a new Blog Author
| **POST** | /cms/v3/blogs/authors/multi-language/attach-to-lang-group | Attach a Blog Author to a multi-language group
| **POST** | /cms/v3/blogs/authors/multi-language/create-language-variation | Create a new language variation
| **POST** | /cms/v3/blogs/authors/multi-language/update-languages | Update languages of multi-language group


## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {hubspotSdkDef} from '@opensdks/sdk-hubspot'

    const hubspot = initSDK(hubspotSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['HUBSPOT_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    // Retrieve a Blog Author
    const res = await hubspot.GET('/cms/v3/blogs/authors/{objectId}')
```
