## Integrating with Hubspot OpenSDK 

Welcome to the Hubspot OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Hubspot OpenSDK allows you to easily integrate with the Hubspot API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt. 

### Available Methods:

| Method | Path | Description |
|--------|------|-------------|
| **POST** | /callback/customer-create/{requestId} | Endpoint for customer creation response
| **POST** | /callback/customer-search/{requestId} | Endpoint for customer search response
| **POST** | /callback/exchange-rate/{requestId} | Endpoint for exchange rate response
| **POST** | /callback/invoice-create/{requestId} | Endpoint for invoice creation response
| **POST** | /callback/invoice-pdf/{requestId} | Endpoint for PDF content of invoice
| **POST** | /callback/invoice-search/{requestId} | Endpoint for invoice search response
| **POST** | /callback/invoices/{requestId} | Endpoint for invoice get-by-id response
| **POST** | /callback/product-search/{requestId} | Endpoint for product search response
| **POST** | /callback/tax-search/{requestId} | Endpoint for taxes search response
| **POST** | /callback/terms/{requestId} | Endpoint for terms search response
| **GET** | /invoice/{invoiceId} | Get invoice data
| **PATCH** | /invoice/{invoiceId} | Update an invoice
| **POST** | /invoice/{invoiceId}/payment | Records an invoice payment
| **GET** | /settings/{appId} | Get URL settings
| **PUT** | /settings/{appId} | Add/Update URL Settings
| **POST** | /sync/{appId}/contacts | Import contacts
| **POST** | /sync/{appId}/products | Import products
| **PUT** | /user-accounts | Create a user account
| **DELETE** | /user-accounts/{accountId} | Delete user account


## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {hubspotSdkDef} from '@opensdks/sdk-hubspot'

    const hubspot = initSDK(hubspotSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['HUBSPOT_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    // Endpoint for customer creation response
    const res = await hubspot.POST('/callback/customer-create/{requestId}')
```
