## Integrating with Clerk OpenSDK 

Welcome to the Clerk OpenSDK documentation. OpenSDKs provides a type-safe and standardized way to interact with various APIs, ensuring a consistent and efficient integration experience.

The Clerk OpenSDK allows you to easily integrate with the Clerk API, providing a robust set of tools to manage your API interactions with managed auth by OpenInt.

## Code Example

```typescript

    import {initSDK} from '@opensdks/runtime'
    import {clerkSdkDef} from '@opensdks/sdk-clerk'

    const clerk = initSDK(clerkSdkDef, {
      auth: {
        // for provider-specific auth
        bearer: `${process.env['CLERK_TOKEN']}`,
        // or for passthrough auth via OPENINT_API_KEY & end user resourceId
        openInt: {
          apiKey: `${process.env['OPENINT_API_KEY']}`,
          resourceId: END_USER_RESOURCE_ID,
        }
      }, params: { ... }
    })

    // Returns the markup for the interstitial page
    const res = await clerk.GET('/public/interstitial')
```
