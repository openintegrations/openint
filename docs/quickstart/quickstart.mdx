## How is OpenInt different from other integration platforms?

- Open Source and entirely focused on authentication
- Launch instantly with our pre-approved Apps and no code integrations portal
- Access hundreds of integrations directly and thousands via aggregator integrations like Plaid, Merge, Finch.
