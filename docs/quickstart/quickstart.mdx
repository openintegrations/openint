# Quickstart Guide

This guide will help you get up and running with OpenInt v1 quickly. We'll cover the basic setup and your first integration.

## Prerequisites

- An OpenInt account
- API credentials
- Node.js 16 or higher

## Installation

```bash
npm install @openint/sdk
# or
yarn add @openint/sdk
```

## Basic Setup

```typescript
import { OpenInt } from '@openint/sdk';

const client = new OpenInt({
  apiKey: 'your-api-key'
});
```

## Creating Your First Connection

```typescript
const connection = await client.connections.create({
  provider: 'salesforce',
  settings: {
    // your connection settings
  }
});
```
