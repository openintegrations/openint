<AccordionGroup>
  <Accordion title="What is OpenInt and why should I use it?">
    OpenInt is an embedded integrations platform designed for quickly deploying product integrations. It reduces the complexity of integration development through no-code or low-code setups and allows seamless data synchronization with your systems.
  </Accordion>

  <Accordion title="How do I integrate OpenInt into my existing application?">
    Integrating OpenInt involves signing up, configuring connectors in the dashboard, installing required packages, and embedding generated tokens or magic links into your app for immediate integration functionality.

    ```bash
    npm install @openint/connect @opensdks/sdk-openint
    ```

  </Accordion>

<Accordion title="What types of integrations does OpenInt support?">
  OpenInt supports a wide range of integrations across various categories such
  as Applicant Tracking Systems (ATS), CRMs, accounting software, and more.
  These can be managed directly from the OpenInt console.
</Accordion>

<Accordion title="How secure is my data with OpenInt?">
  OpenInt prioritizes data security by enabling direct synchronization with your
  own database, significantly minimizing data exposure and enhancing privacy.
</Accordion>

  <Accordion title="How can OpenInt streamline user onboarding and integration management?">
    OpenInt provides magic links—secure URLs—to simplify user onboarding and integration setup. These links facilitate quick sharing and embedding, enhancing user experience and reducing integration friction.
  </Accordion>
</AccordionGroup>
