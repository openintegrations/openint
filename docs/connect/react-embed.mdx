# React Embed

OpenInt provides a React component library that makes it easy to embed our connection flow into your application.

## Installation

```bash
npm install @openint/react
# or
yarn add @openint/react
```

## Basic Usage

```tsx
import { OpenIntConnect } from '@openint/react';

function App() {
  return (
    <OpenIntConnect
      apiKey="your-api-key"
      onSuccess={(connection) => {
        console.log('Connection created:', connection.id);
      }}
      onError={(error) => {
        console.error('Connection failed:', error);
      }}
    />
  );
}
```

## Customization

```tsx
<OpenIntConnect
  apiKey="your-api-key"
  theme={{
    colors: {
      primary: '#6200ff',
      secondary: '#07C983'
    },
    borderRadius: '8px'
  }}
  providers={['salesforce', 'hubspot']}
  metadata={{
    userId: '123',
    teamId: '456'
  }}
/>
```

## Props Reference

- `apiKey` (required) - Your OpenInt API key
- `onSuccess` - Callback when connection is successful
- `onError` - Callback when connection fails
- `theme` - Custom styling options
- `providers` - Array of enabled providers
- `metadata` - Custom metadata to include

## Best Practices

- Handle errors appropriately
- Implement proper loading states
- Use appropriate event handlers
- Follow React best practices
- Test thoroughly
