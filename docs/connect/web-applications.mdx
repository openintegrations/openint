# Web Applications

OpenInt Connect can be integrated into any web application using our JavaScript SDK.

## Installation

```html
<script src="https://cdn.openint.com/connect.js"></script>
```

Or using npm:

```bash
npm install @openint/connect
# or
yarn add @openint/connect
```

## Basic Implementation

```javascript
import {OpenIntConnect} from '@openint/connect'

const connect = new OpenIntConnect({
  apiKey: 'your-api-key',
})

// Open the connect modal
connect.open({
  onSuccess: (connection) => {
    console.log('Connection created:', connection.id)
  },
  onError: (error) => {
    console.error('Connection failed:', error)
  },
})
```

## Configuration Options

```javascript
connect.open({
  providers: ['salesforce', 'hubspot'],
  theme: {
    colors: {
      primary: '#6200ff',
      secondary: '#07C983',
    },
  },
  metadata: {
    userId: '123',
    teamId: '456',
  },
  mode: 'modal', // or 'redirect'
  redirectUrl: 'https://your-app.com/callback',
})
```

## Event Handling

```javascript
connect.on('connection.created', (connection) => {
  // Handle new connection
})

connect.on('connection.failed', (error) => {
  // Handle connection error
})

connect.on('modal.closed', () => {
  // Handle modal close
})
```

## Best Practices

- Initialize the SDK only once
- Handle all possible events
- Implement proper error handling
- Use appropriate security measures
- Test across different browsers
